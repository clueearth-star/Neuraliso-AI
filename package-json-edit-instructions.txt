EDIT TO package.json
--------------------
Do NOT replace your whole package.json — just add ONE line.

Open package.json in GitHub's editor, find the "scripts" section
(it will look something like this):

  "scripts": {
    "dev": "tsx server.ts",
    "build": "vite build && esbuild server.ts --bundle --platform=node --format=cjs --packages=external --sourcemap --outfile=dist/server.cjs",
    "start": "node dist/server.cjs",
    ...
  },

Add this new line directly after the "build" line (don't forget the comma
at the end of the "build" line above it, and after this new line too if
more lines follow):

    "vercel-build": "vite build",

Result should look like:

  "scripts": {
    "dev": "tsx server.ts",
    "build": "vite build && esbuild server.ts --bundle --platform=node --format=cjs --packages=external --sourcemap --outfile=dist/server.cjs",
    "vercel-build": "vite build",
    "start": "node dist/server.cjs",
    ...
  },

Then commit.
